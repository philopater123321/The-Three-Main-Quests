# The three main Quests Project
### Languages Used: JavaScript, HTML, and CSS.
### Description of the Project:
This project is made for productive people, as it forces them to do only 3 quests or tasks per day.
It has many features, such as inserting tasks, a timer, and a quote section.
### Why are these sections there?
I wanted to make this to decrease the distraction, as you will not need to go to many tabs, one for the timer and another for quotes to re-motivate you.
Instead, I chose to insert all of these in only one project to make the user's life easier.
When you finish all your tasks, the program congratulates you and advises you to take a break.
